California MCA pipeline (v2.0)
Stage A — Ingest (Bulk files)

In bizfile Online, request UCC bulk data (full snapshot + periodic deltas). 
California Secretary of State

You’ll receive ZIPs containing Debtors / Secured Parties (and often Filing/Action tables).

In Replit, mount/upload the ZIPs (or sync from your cloud) and extract.

If/when California adds or changes bulk formats, this design keeps working because we don’t hardcode field names—we pick from a list of aliases per field.

Stage B — Score (business roll-up)

Exactly like Colorado, tuned for CA lender landscape:

Positives

has_bank: tokens like “ bank”, “credit union”, “ national association”, “ n.a”, “ fsb”, “savings bank”.
(Think CA lenders you’ll see a lot: Bank of America, Wells Fargo, U.S. Bank, JPMorgan, Zions, First Citizens/CIT, City National, etc.)

has_equipment: tokens like john deere, caterpillar, CNH, Kubota, DLL, Komatsu, Volvo Financial, Vermeer, Doosan, plus CA-common: Key Equipment Finance, PNC Equipment Finance, Banc of California Equipment Finance, Toyota Industries/TCF legacy, Balboa Capital, Leaf Capital.

Negatives: has_irs, has_sba, and public-sector names (e.g., “City of ”, “County of ”, “Unified School District”, “Authority”, “University”, “State of California”).

Counts: active_filing_count, secured_party_count.

Recency (if the file/action date is included in CA bulk—many bundles include it).

Score (weights can live in YAML as before):

score =
  +25*has_bank   +25*has_equipment
  +20*min(secured_party_count,10)/10
  +20*min(active_filing_count,10)/10
  +10*recency_bucket   # if CA data includes filed/action date
  -60*has_irs          -40*has_sba


Deduping: normalize (name, zip) + fuzzy collapse (RapidFuzz). Suppress entities previously exported (SQLite “seen” table).

2) main_ca.py (drop-in; same structure as the CO script—but reads local CA bulk CSVs instead of calling an API)

Changes vs. CO:

load_csvs(glob) → stream multiple CSVs (some CA bulks split by year/month).

Robust column pickers: fileid|filenumber|fileno|recordnumber, organizationname|orgname, lastname|firstname, address1|addressline1|address, zipcode|zip|postalcode, recordstatus|recordstatuscd|status, filedate|filingdate|recorddate|eventdate.

If you have a filings/actions table with dates keyed by fileid, left-join to populate last_filing_date.

I’m keeping the code payload compact here; you can reuse the CO main.py I gave you and replace the Socrata “pull” with a tiny “load from glob” function:

# snippet you’ll paste into your existing main.py, replacing the Socrata pull
import glob

def load_many(glob_pattern, nrows=None):
    dfs = []
    for path in glob.glob(glob_pattern):
        try:
            dfs.append(pd.read_csv(path, dtype=str, nrows=nrows, encoding="utf-8", low_memory=False))
        except UnicodeDecodeError:
            dfs.append(pd.read_csv(path, dtype=str, nrows=nrows, encoding="latin-1", low_memory=False))
    return norm_cols(pd.concat(dfs, ignore_index=True)) if dfs else pd.DataFrame()

def pull_debtors_ca():
    g = cfg["california_bulk"]["debtor_glob"]
    return load_many(g)

def pull_secured_ca():
    g = cfg["california_bulk"]["secured_glob"]
    return load_many(g)

def pull_filings_ca():
    g = cfg["california_bulk"].get("filings_glob")
    return load_many(g) if g else pd.DataFrame()

# then in main(): use pull_debtors_ca(), pull_secured_ca(), and optionally pull_filings_ca()

Tips that boost quantity & quality in California

Use the deltas: after the full snapshot, request weekly/monthly updates from BE & UCC Bulk Orders, then append + re-score. 
California Secretary of State

Heavier lender dictionary: start with the CA-heavy list in terms.equipment and expand as you see new secured parties in the data; each time you add a token, your equipment-positive recall increases.

Recency if available: many CA bulk bundles include an action/filing date—if you’ve got it, give +recency weight for last 12–24 months.

Sectors that convert: prioritize horizontal construction, paving, grading, trenching, pipeline, drilling, electrical, mechanical, concrete, aggregates, materials, trucking—then let outcomes fine-tune weights.

Blacklist public sector early (school districts/cities/counties/authorities), and suppress IRS/SBA/FTB tokens.
